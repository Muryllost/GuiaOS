Veredito Acadêmico

Para uso Doméstico?

O Windows é o rei da compatibilidade para jogos e softwares comuns. Mas o ChromeOS e Mint são perfeitos e gratuitos para ressuscitar computadores antigos onde o Windows trava.

Para Empresas e Servidores?

No escritório: Windows ou macOS devido ao pacote Office e Adobe. Nos Servidores de internet: Os sistemas livres (Linux) dominam pela segurança extrema e custo zero de licença.

Sistemas Livres podem substituir?

Sim, e devem! Em governos, escolas e para desenvolvedores de software, o uso do Linux poupa milhões em dinheiro público e oferece o mesmo nível (ou superior) de qualidade.

A porta de entrada ideal. O design (Cinnamon) simula o menu iniciar do Windows 7/10.

Voltado 100% para internet. Revive notebooks antigos perfeitamente em escolas.

Comandos Linux

  • ls : Listar arquivos
  • pwd : Mostrar pasta atual
  • ifconfig : Dados de rede/IP
  • whoami : Usuário logado
  • clear : Limpar tela

Comandos Windows

  • dir : Listar arquivos
  • cd : Mostrar pasta atual
  • ipconfig : Dados de rede/IP
  • whoami : Usuário logado
  • cls : Limpar tela
