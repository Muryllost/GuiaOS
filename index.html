O que são esses sistemas?

Sistemas Pagos / Proprietários

Pense num Restaurante Chique. Você paga pela refeição, senta-se na mesa e o prato chega pronto e delicioso. É fácil e confortável.

O detalhe: A porta da cozinha está trancada. Você não pode ver a receita, não pode mudar os ingredientes e, se quiser comer de novo, tem que pagar outra vez. Isso é o Código FechadoSoftware em que apenas a empresa que o criou pode ver ou modificar como ele foi programado..

Sistemas Livres / Código Aberto

Pense numa enorme Cozinha Comunitária. Os ingredientes (o sistema) são totalmente gratuitos. Você pode entrar, cozinhar, ver o livro de receitas e até criar um prato novo.

O detalhe: Como você é o chef, vai precisar aprender a usar o fogão e as panelas. Se a comida queimar, a comunidade te ajuda a limpar, mas não há um "garçom" oficial. Isso é o Código Aberto (Open Source)Software em que o código de programação é público. Qualquer pessoa no mundo pode ver, estudar e melhorar..

Qual é o mais indicado para uso doméstico?

Para a maioria, o Windows é o rei do conforto doméstico. Quase todos os jogos e programas escolares funcionam nele sem esforço. No entanto, o ChromeOS Flex ou o Linux Mint são alternativas fantásticas e gratuitas para ressuscitar computadores antigos onde o Windows trava, oferecendo uma experiência focada na internet com zero chance de pegar vírus comuns.

E para empresas e servidores?

Para os funcionários de escritório, o Windows ou macOS dominam pela compatibilidade com o Pacote Office e Adobe. Já nos Servidores (os computadores invisíveis que mantêm a internet funcionando), os Sistemas Livres (Debian, Ubuntu Server) dominam absolutamente o mundo. Eles são leves, incrivelmente seguros, não exigem interface gráfica e não cobram licenças milionárias por uso.

Sistemas livres podem substituir os pagos?

Sim, e cada vez mais! Para desenvolvimento de software, navegação web, administração pública (escolas e governos), o Software Livre não só substitui como supera os pagos, gerando economia brutal de dinheiro público. A substituição só é difícil em nichos muito específicos, como edições de cinema (Adobe Premiere) ou jogos com sistemas anti-trapaça muito restritos que só existem para o Windows.
